npx:supersedes: https://w3id.org/np/RASsLPDBpyeofCHRTdy2NYLtU-mep--kB7eRzIHFPkawo

    A visualization for the RDF vocabulary that groups terms for properties, triples, containers, language values, compound literals, and values. Iolanta picks this nanopub up via iolanta:visualizes rdf: and renders the grouped terms when the user navigates to the rdf: namespace.

    Properties

    • Property

      The class of RDF properties.

    • type

      The subject is an instance of a class.

    Triples

    • Statement

      The class of RDF statements.

    • object

      The object of the subject RDF statement.

    • predicate

      The predicate of the subject RDF statement.

    • subject

      The subject of the subject RDF statement.

    Containers

    • Alt

      The class of containers of alternatives.

    • Bag

      The class of unordered containers.

    • List

      The class of RDF Lists.

    • Seq

      The class of ordered containers.

    • first

      The first item in the subject RDF list.

    • nil

      The empty list, with no items in it. If the rest of a list is nil then the list has no more items in it.

    • rest

      The rest of the subject RDF list after the first item.

    Language

    • langString

      The datatype of language-tagged string values

    • language

      The language component of a CompoundLiteral.

    Compound Literal

    • CompoundLiteral

      A class representing a compound literal.

    • direction

      The base direction component of a CompoundLiteral.

    Value

    • value

      Idiomatic property used for structured values.

    Datatype

    • HTML

      The datatype of RDF literals storing fragments of HTML content

    • JSON

      The datatype of RDF literals storing JSON content.

    • PlainLiteral

      The class of plain (i.e. untyped) literal values, as used in RIF and OWL 2

    • XMLLiteral

      The datatype of XML literal values.

    • langString

      The datatype of language-tagged string values
